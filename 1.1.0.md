# New Features

> ### **Release Highlights**
> 
> - Added Azure CSP support
> - Added My Page
> - Improved snapshot-based restore

# New Features

## Azure CSP Support

All features available in OwlDB 1.0.0 are now also available on Azure.

{% hint style="info" %}
**Note**
Archive restore is not supported on Azure.
{% endhint %}

## Dashboard

### **Top N Chart**

A Top N chart has been added to the dashboard. You can view information for up to the top 5 instances with the highest CPU usage, memory usage, and session load.

## Estimated Cost

When infrastructure resource configurations are added or changed, such as during provisioning or spec changes, you can check the estimated cost for the database.

{% hint style="info" %}
**Note**

Actual costs may vary depending on multiple factors such as the region and the user’s workload.
{% endhint %}

## My Page

### **Account Management**

A feature has been added that allows you to view information about the account currently in use and change the password.

### **Usage Management**

A feature has been added that allows you to view license charges for each database managed in OwlDB.

- In the Usage Status menu, you can check the current month’s database usage status.
- In the Usage History menu, you can check monthly database usage history for up to the last 5 years.

{% hint style="info" %}
**Note**

Infrastructure resource costs are not included.
{% endhint %}

## Notifications

### **Email Delivery**

An email delivery feature has been added. You can now also receive various notifications provided by OwlDB by email.

# **Changes**

## Feature Improvements

### Filter Settings

Improved an issue where the selected database or instance in the Management and Monitoring menus was reset when the page was refreshed.

### Snapshot-Based Restore

The restore behavior has been improved to make the feature more intuitive.

- If you perform a restore without selecting a snapshot, you are prompted to enter a restore point, and the database is restored to that point, as before.
- If you perform a restore after selecting a snapshot, the database is restored immediately to the point in time of the selected snapshot.

## Usability Improvements

### Menu Structure

The overall OwlDB menu structure has been changed. The previous 1-depth side menu has been replaced with a 2-depth side menu.

### List View Method

Components that were previously provided with infinite scroll have been changed to use pagination.
