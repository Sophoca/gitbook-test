# 1.1.0

> ### **Release Highlights**
> 
> - Azure CSP Added
> - My Page Added
> - Snapshot-Based Recovery Improvement

# New Features

## Azure CSP Added

All features provided by OwlDB 1.0.0 are now available on Azure.

{% hint style="info" %}
**Note**
The archive recovery feature is not supported on Azure.
{% endhint %}

## Dashboard

### **Top N Chart**

A TopN chart has been added to the dashboard. You can view information for up to the top 5 instances with the highest CPU usage, Memory usage, and Session load.

## Estimated Cost

When infrastructure resource configurations such as provisioning and spec changes are added or modified, you can view the estimated cost based on the database.

{% hint style="info" %}
**Note**

Actual costs may vary depending on multiple factors such as region and user workload.
{% endhint %}

## My Page

### **Account Management**

A feature has been added to view information for the currently used account and change the password.

### **Usage Management**

A feature has been added to look up license fees per database managed in OwlDB.

- In the Usage Status menu, you can view the current month's database usage status.
- In the Usage History menu, you can view monthly database usage history for up to the last 5 years.

{% hint style="info" %}
**Note**

Infrastructure resource costs are excluded.
{% endhint %}

## Notifications

### **Email Delivery**

An email delivery feature has been added. You can now receive the various notifications provided by OwlDB via email as well.

---

# **Changes**

## Feature Improvements

### Filter Settings

Fixed an issue where the database/instance selected in the management and monitoring menus would be reset when refreshing the page.

### Snapshot-Based Recovery Feature

The behavior has been improved to increase the intuitiveness of the recovery feature.

- If you perform recovery without selecting a snapshot, you separately enter the target point in time to recover to, as before.
- If you perform recovery after selecting a snapshot, recovery to the point in time of the selected snapshot proceeds immediately.

## Usability Improvements

### Menu Structure

The overall menu structure of OwlDB has been changed. The navigation has been updated from a 1-depth side menu to a 2-depth side menu.

### List View Method

Components that were previously provided with infinite scroll have been changed to pagination.
