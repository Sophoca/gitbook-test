# Feature Usage Guide / Alerts

Provides notifications for various events that occur while using OwlDB. Notifications can be viewed by clicking the 🔔 icon in the top-right corner of the console screen.

- To the right of the message, :2714:click the icon to freely toggle the read status.
- Messages marked as read are automatically deleted after 30 days.
- Click the meatball menu icon button in the top-right corner > '**Mark All as Read**' to mark all messages as read.
