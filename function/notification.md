# Alerts

Provides notifications for various events that occur while using OwlDB. Notifications can be viewed by clicking the 🔔 icon in the upper right corner of the console.

- To the right of a message, click the :2714: icon to freely toggle the read status.
- Messages marked as read are automatically deleted after 30 days.
- Click the meatball menu icon button in the upper right > '**Mark All as Read**' to mark all messages as read.
