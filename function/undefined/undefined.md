# Login

Once the OwlDB installation is complete, the PS engineer will provide an access link. Navigating to that link will take you to the login page.

## Root initial login

The default username and password for the Root account are as follows.

| Field | Default value |
| --- | --- |
| Username | admin |
| Password | admin |

When you log in with the initial password, you will be automatically redirected to the password change page for security purposes. After setting a new password and logging in again, you will be successfully connected with the Root account.

{% hint style="warning" %}
**Caution**

You must change your password after the initial login.

You cannot navigate to any other page until the password has been changed.
{% endhint %}

---

## Login

1. Enter your username and password on the login page.
2. **[Login]** Click the button.
3. Once authentication is complete, you will be redirected to the dashboard page.

{% hint style="info" %}
**Note**

If you do not have a Member account, on the login page, **[Request Account Creation]** click the button to submit a request to create an account. For more information, refer to Request Account Creation.
{% endhint %}
